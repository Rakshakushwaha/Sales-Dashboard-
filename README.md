# Sales-Dashboard-

Repository description for your Madhav Ecommerce Sales Dashboard project:

1. Project Title / Headline

   📊 Madhav Ecommerce Sales Dashboard | Sales Performance & Profit Analysis using Power BI

2. Short Description / Purpose

   The Madhav Ecommerce Sales Dashboard is an interactive Power BI project designed to analyze sales performance, profit trends, customer behavior, product categories, and payment preferences. The dashboard provides actionable business insights that help stakeholders monitor revenue, identify profitable products, and make data-driven decisions to improve overall business performance.

3. Tech Stack

   Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel
Data Modeling
Data Visualization

5. Data Source

    Ecommerce Sales Dataset (Excel/CSV)
Customer Information
Product Categories & Sub-Categories
Sales Transactions
Profit Data
State-wise Sales Records
Payment Mode Details

7. Features / Highlights

   🔹 Interactive Ecommerce Sales Dashboard

    Key Insights:
• Total Sales Amount, Profit, Quantity Sold, and Average Order Value (AOV)
• State-wise Sales Performance Analysis
• Category-wise Quantity Distribution
• Monthly Profit Trend Analysis
• Customer-wise Revenue Contribution
• Payment Mode Analysis (COD, UPI, Debit Card, Credit Card, EMI)
• Sub-Category Profit Performance
• Quarterly and State Filters for Dynamic Analysis

      Business Value:
• Track overall business performance in real time
• Identify top-performing states and customers
• Analyze profitable product categories and sub-categories
• Understand customer payment preferences
• Support strategic sales and inventory decisions
